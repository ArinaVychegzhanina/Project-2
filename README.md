# Проект по дисциплине "Языки и методы программирования"
Покупка коттеджей с учётом заданных параметров

# Описание проекта
Данный проект подходит людям, которые желают приобрести коттедж, соответствующий необходимым параметрам. Кроме того, простой и удобный интерфейс позволит любому человеку без проблемм подобрать себе варианты коттеджей.
